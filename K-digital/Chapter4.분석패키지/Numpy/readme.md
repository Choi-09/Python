## Numpy

1. Numpy란?
  + 데이터각분야에서 사용되는 판다스,싸이파이, 맷플로립 등과 함께 머신러닝, 딥러닝 운용의 기반이 되는 오픈소스 라이브러리.
  
2. 배열
  ```
  - 배열이란: 여러 원소들의 집합
  - n차열 배열(demension array): nd array (n차원에 따라서 구분한 배열의 형태)
    ex)
    1차원 배열: [1,2,3,4]
    
    2차원 배열: [[1,2,3],
                [4,5,6]]  : 0행 2열 = 3, 1행 1열 = 5
   
   3차원 배열: [ [1,2,3],
                  [4,5,6],
                  [7,8,9] ],
                  [ [1,2,3],
                  [4,5,6],
                  [7,8,9] ],
                 [ [1,2,3],
                  [4,5,6],
                  [7,8,9] ]
  ```
